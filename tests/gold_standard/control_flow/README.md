# Control Flow: if/else, while, for, break, continue

Tests for branching and looping constructs. Covers `if`/`else` branches, `while` loops with mid-loop condition updates, and `for i in 0..N` ranges. Includes nested conditionals and loops with early-exit / sentinel patterns.

## What belongs here

- for-range loops with accumulator update
- while loops with three-way branch (binary search)
- if/else if/else chains
- Loop termination via sentinel comparison
